📖 Що таке BudgUA

BudgUA — це персональний інструмент для планування та контролю сімейного бюджету. Сервіс допомагає:

💰
Планувати бюджет
Доходи, витрати, кредити, розстрочки та кредитні картки в одному місці
📱
Telegram-бот
Додавайте витрати фото чеку або текстом прямо з телефону
🤖
AI розпізнавання
Gemini автоматично витягує суму, категорію та дату з фото чека
📊
Аналітика
Місячна та річна статистика, прогноз погашення боргів

🔐 Реєстрація та вхід

1
Відкрийте сайт BudgUA
Перейдіть на головну сторінку. Оберіть мову — UA або ENG.
2
Зареєструйтесь
Натисніть «Реєстрація», введіть ім'я, email та пароль (мін. 6 символів).
3
Увійдіть до кабінету
Після входу ви потрапляєте на головний Дашборд.

📋 Створення бюджету

Бюджет — основна одиниця планування. Зазвичай створюється на місяць або квартал.

1
Перейдіть до «Бюджети»
У боковому меню → «Бюджети» → кнопка «+ Новий бюджет».
2
Заповніть форму
Назва, місяць, загальний дохід.
3
Додайте борги (якщо є)
Кредити, розстрочки, кредитні картки — з сумою, ставкою та платежем.
4
Вкажіть очікувані витрати
Категорії з плановими сумами: продукти, транспорт тощо.

Поля бюджету

Поле Опис
Назва Наприклад: «Червень 2025» або «Сім'я Q2»
Дохід Загальний місячний дохід сім'ї (всі джерела)
Кредити Назва, повна сума, залишок, платіж/міс, місяців до кінця
Розстрочки Аналогічно кредитам — для товарів у розстрочку
Кредитні картки Борг, річна ставка, ваш платіж. Автоматично рахує: відсотки, мін. платіж (5%), погашення боргу
Категорії Планові витрати по категоріях для відстеження факту
💡 Порада: Вкажіть «Повну суму» кредиту — тоді в дашборді з'явиться прогрес-бар погашення.

📊 Дашборд

Головний екран показує поточний стан бюджету в реальному часі:

  • Дохід — загальний місячний дохід
  • Реальний залишок — дохід мінус всі виплати та фактичні витрати
  • Витрачено (факт) — сума транзакцій з Telegram бота та ручних записів
  • Борг / До погашення — загальна сума та приблизна дата закриття
  • Категорії: факт vs план — прогрес-бари по кожній категорії
  • Останні транзакції — останні 5 витрат з позначкою джерела
  • Борги та кредити — таблиця з деталями та датами погашення
🟢 Зелена точка «Online» в сайдбарі означає, що Telegram бот підключений і дані оновлюються в реальному часі.

📅 Статистика за рік

Перейдіть у меню «📅 Рік» для перегляду помісячної статистики:

  • Стовпчики: зелений = місяць в плюсі, червоний = дефіцит
  • Таблиця: дохід, витрати, баланс, кількість транзакцій по місяцях
  • Клік на стовпець → деталі місяця з розбивкою по категоріях

🤖 Підключення Telegram бота

Бот: @BudgUA_bot
1
Отримайте API токен
Зайдіть на сайт → Кабінет → скопіюйте токен із блоку «Telegram бот».
2
Відкрийте бот
В Telegram знайдіть @BudgUA_bot або натисніть /start.
3
Натисніть «🔑 Підключити акаунт»
Кнопка з'явиться в меню після запуску бота.
4
Вставте токен
Бот попросить ввести токен — вставте скопійований з сайту. Після успішного підключення бот надішле підтвердження.
⚠️ Не передавайте API токен стороннім особам. У разі компрометації — оновіть токен у Кабінеті.

Використання бота

📋
Обрати бюджет
Кнопка «📋 Обрати бюджет» — встановлює бюджет за замовчуванням. При одному бюджеті — встановлюється автоматично.
💸
Додати витрату
Кнопка «💸 Додати витрату» — переходить в режим введення. Надішліть фото або текст. Кнопка «↩ Назад» виходить з режиму.
📸
Фото чека
Просто надішліть фото чека — AI (Gemini) розпізнає суму, категорію, магазин і дату автоматично.
✍️
Текст
Напишіть витрату текстом: Продукти АТБ 450 або Кава 85 грн
📊
Статистика
Кнопка «📊 Статистика» — показує підсумок місяця: витрати, баланс, план vs факт, борги.
🌐
Мова
Кнопки UA / ENG перемикають мову бота.

🔑 Gemini API Key

BudgUA використовує Google Gemini для розпізнавання чеків. Ви можете використовувати власний безкоштовний ключ.

Як отримати ключ

1
Перейдіть на Google AI Studio
aistudio.google.com/app/apikey — потрібен Google акаунт.
2
Натисніть «Create API key»
Оберіть або створіть проєкт. Ключ буде згенеровано одразу.
3
Скопіюйте ключ
Він виглядає як AIzaSy... — збережіть його.
4
Додайте в BudgUA Кабінет
Кабінет → блок «AI ключі» → вставте ключ → «💾 Зберегти».
💡 Безкоштовний план Google Gemini включає достатню кількість запитів для особистого використання.

Система fallback (запасні моделі)

BudgUA автоматично перемикається між моделями при помилці:

  1. Власний ключ → gemini-2.5-flash
  2. Власний ключ → gemini-1.5-flash
  3. Власний ключ → gemini-1.0-pro
  4. Системний ключ → gemini-2.5-flash
  5. Системний ключ → gemini-1.5-flash
Це гарантує, що навіть при вичерпанні ліміту вашого ключа або тимчасовій недоступності моделі — розпізнавання чеків продовжить працювати.

📄 PDF звіти

На дашборді натисніть «📄 Звіт PDF». Оберіть мову звіту (UA/ENG).

Звіт містить: загальний підсумок, прогрес погашення боргів, графік витрат по категоріях, таблиці кредитів та карток.

Звіт відкривається у новому вікні браузера. Натисніть Ctrl+P або «Друк» → «Зберегти як PDF».

❓ FAQ

Чек не розпізнається

Переконайтеся що фото чіткі та рівні. Спробуйте ввести витрату текстом. Перевірте ліміти Gemini API.

Як видалити акаунт?

Кабінет → Небезпечна зона → «🗑 Видалити акаунт». Потрібно підтвердити двічі ввівши DELETE. Дія незворотна.

Скільки бюджетів можна створити?

Необмежено. Рекомендується один бюджет на місяць або квартал.

📖 About BudgUA

BudgUA is a personal family budget planning and tracking tool that combines a web interface, Telegram bot, and AI receipt scanning in one product.

💰
Budget Planning
Income, expenses, loans, installments and credit cards in one place
📱
Telegram Bot
Add expenses via receipt photo or text message from your phone
🤖
AI Recognition
Gemini automatically extracts amount, category and date from receipt photos
📊
Analytics
Monthly and yearly statistics, debt payoff forecast

🔐 Registration

1
Open BudgUA
Go to the main page. Choose language — UA or ENG.
2
Register
Click «Sign up», enter name, email and password (min. 6 characters).
3
Access your profile
After login you'll land on the Dashboard.

🤖 Connecting Telegram Bot

Bot: @BudgUA_bot
1
Get your API token
Go to website → Profile → copy token from «Telegram bot» block.
2
Open the bot
Find @BudgUA_bot in Telegram or click /start.
3
Tap «🔑 Connect account»
The button appears in the menu after launching the bot.
4
Paste your token
The bot will ask for your token — paste the one from the website. You'll receive a confirmation message.

🔑 Gemini API Key

1
Go to Google AI Studio
aistudio.google.com/app/apikey
2
Click «Create API key»
Select or create a project. The key will be generated immediately.
3
Copy the key
It looks like AIzaSy...
4
Add to BudgUA Profile
Profile → «AI Keys» block → paste key → «💾 Save».
